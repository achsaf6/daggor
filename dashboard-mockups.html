Daggor Dashboard — design comparison
1 · Director's Desk refined theatrical · low risk
2 · Field Command tactical terminal · high density
3 · Glass Atelier translucent · map-first
4 · Hand-bound Codex illustrative · atmospheric

Director's Desk — refined theatrical

Same theme, 60% less ornament. Brass hairlines instead of full borders, smaller titles, drag handles and badges only on hover. The map breathes.

Vibe

Polished tabletop game screen. Quiet. Apothecary's ledger.

Trade-off

Lowest risk. Doesn't help if you're tired of the aesthetic itself.

Field Command — tactical terminal

Charcoal, mono, amber accent. Every column aligns. Active combatant inverts to amber. Player roster is a 2-col card grid with live/off heat bars. Soundboard is a horizontal strip across the bottom.

Vibe

Mission control. Fast combat ops. Linear × Foundry × NORAD.

Trade-off

Loses the storybook feel entirely. Looks like a sysadmin's screen.

Glass Atelier — translucent floating

Map fills the whole viewport. Panels are blurred glass, fade when idle. Initiative collapses to a horizontal "now / next" strip. Players collapse to a stack of color dots with names on hover. Toolbar is a vertical icon ribbon.

Vibe

visionOS / Arc browser. Modern, premium, map-dominant.

Trade-off

Glass fights with busy battlemap textures. Doesn't read as "tabletop".

Hand-bound Codex — illustrative

Layered parchment with grain. Players are wax seals (color = wax). Initiative entries use roman numerals; the active combatant gets a quill icon. Soundboard knobs are brass dials. Tool icons read like brass instruments.

Vibe

Baldur's Gate 3 / Pillars of Eternity / illuminated manuscript.

Trade-off

Highest atmosphere, lowest density. 8+ combatants gets cramped.
